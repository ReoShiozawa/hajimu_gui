# hajimu_gui

**はじむ言語用 GUI パッケージ v13.0.0** — 自作プラットフォーム層 + 即時モード GUI フレームワーク

egui のシンプルさと PyQt の本格ウィジェットを融合した、**1,074関数**搭載の完全日本語 API GUI ライブラリです。

---

## 特徴

- **即時モード GUI** — `GUI.ボタン("ラベル")` が `真/偽` を返すシンプル設計
- **完全日本語 API** — 関数名・引数名すべて日本語
- **高品質描画** — 自作レンダラ (hjp_render.c) によるアンチエイリアス付きベクター描画
- **日本語フォント内蔵** — Noto Sans CJK JP を同梱（追加設定不要）
- **Retina 対応** — macOS の高解像度ディスプレイに自動対応
- **依存ゼロ** — Qt/GTK/SDL2 不要、自作プラットフォーム層のみ
- **クロスプラットフォーム** — macOS (Cocoa) / Linux (X11+GLX) / Windows (Win32+Win64)
- **ダークテーマ** — モダンなダーク UI をデフォルト搭載
- **単一ファイル構成** — hajimu_gui.c 1ファイルで全1,074関数を実装

## セットアップ

### 必要な依存

**外部依存ライブラリは不要です。** OS標準のグラフィックスAPI のみ使用します。

| OS | 必要なもの |
|---|---|
| macOS | Xcode Command Line Tools (`xcode-select --install`) |
| Linux | X11 + OpenGL (`sudo apt install libx11-dev libgl-dev`) |
| Windows | Visual Studio / MinGW (Win32 API) |

### ビルド & インストール

```bash
git clone https://github.com/ReoShiozawa/hajimu_gui.git
cd hajimu_gui
make
make install
```

## 使い方

```
取り込む "hajimu_gui" として GUI

// アプリ作成
変数 アプリ = GUI.アプリ作成("私のアプリ", 800, 600)

// 状態
変数 カウンター = 0
変数 同意 = 偽

// 描画ループ
GUI.描画ループ(アプリ, 関数():
    GUI.見出し("Hello, はじむ GUI!", 1)
    GUI.テキスト("即時モードGUIのデモです")
    GUI.セパレーター()

    もし GUI.ボタン("カウント +1") なら
        カウンター = カウンター + 1
    終わり
    GUI.テキスト("カウント: " + 文字列(カウンター))

    同意 = GUI.チェックボックス("利用規約に同意する", 同意)
終わり)

GUI.アプリ終了(アプリ)
```

## API リファレンス

### Phase 1: コア (v0.1.0)

| 関数 | 引数 | 戻り値 | 説明 |
|---|---|---|---|
| `アプリ作成` | タイトル, 幅, 高さ | 数値 | ウィンドウ付きアプリを作成 |
| `描画ループ` | アプリ, コールバック | — | メインループ開始 |
| `アプリ終了` | アプリ | — | アプリを終了・リソース解放 |
| `ウィンドウサイズ` | アプリ | {幅, 高さ} | 現在のウィンドウサイズ |
| `ウィンドウタイトル` | アプリ, タイトル | — | タイトルバー変更 |
| `ウィンドウリサイズ` | アプリ, 幅, 高さ | — | ウィンドウサイズ変更 |
| `フレームレート` | アプリ, FPS | — | 目標FPS設定 (デフォルト: 60) |
| `背景色` | 色 | — | 背景色設定 (パック色 or 16進文字列) |
| `色` | R, G, B [, A] | 数値 | RGBA色を生成 (0–255) |
| `色16進` | "#RRGGBB" | 数値 | 16進数から色を生成 |

### Phase 2: ウィジェット (v0.2.0)

| 関数 | 引数 | 戻り値 | 説明 |
|---|---|---|---|
| `テキスト` | 内容 [, オプション] | — | テキスト描画（折り返し対応） |
| `見出し` | 内容 [, レベル] | — | 見出し (h1–h6) |
| `ボタン` | ラベル [, オプション] | 真偽 | クリックで `真` を返す |
| `チェックボックス` | ラベル, 値 | 真偽 | トグル状態を返す |
| `ラジオボタン` | ラベル, グループ, 現在値 | 文字列 | 選択されたラベルを返す |
| `スライダー` | ラベル, 値, 最小, 最大 | 数値 | ドラッグで値変更 |
| `プログレスバー` | 値 [, 最大] | — | 進捗表示 (デフォルト最大: 100) |
| `セパレーター` | — | — | 水平区切り線 |
| `スペーサー` | [高さ] | — | 余白 (デフォルト: 16px) |
| `ツールチップ` | テキスト | — | 直前ウィジェットのホバーヒント |

### Phase 3: テキスト入力 (v0.3.0)

| 関数 | 引数 | 戻り値 | 説明 |
|---|---|---|---|
| `テキスト入力` | ラベル, 値 [, オプション] | 文字列 | 単行テキスト入力 |
| `パスワード入力` | ラベル, 値 | 文字列 | マスク付き入力 (●表示) |
| `数値入力` | ラベル, 値 [, 最小, 最大, ステップ] | 数値 | ±ボタン付き数値入力 |
| `テキストエリア` | ラベル, 値 [, 行数] | 文字列 | 複数行テキスト入力 |
| `コンボボックス` | ラベル, 選択肢, 選択値 | 値 | ドロップダウン選択 |

### Phase 4: レイアウト (v0.4.0)

| 関数 | 引数 | 戻り値 | 説明 |
|---|---|---|---|
| `パネル開始` | [タイトル, オプション] | — | コンテンツグループ開始 |
| `パネル終了` | — | — | パネルを閉じる |
| `横並び開始` | — | — | 横並びレイアウト開始 |
| `横並び終了` | — | — | 横並びレイアウト終了 |
| `インデント` | [幅] | — | インデント追加 (デフォルト 20px) |
| `インデント解除` | — | — | インデントを解除 |
| `マウス位置` | — | {x, y} | マウスの現在位置 |
| `マウスクリック` | — | 真偽 | このフレームでクリックされたか |

> 全1,074関数の完全なリファレンスは [docs/REFERENCE.md](docs/REFERENCE.md) を参照してください。
